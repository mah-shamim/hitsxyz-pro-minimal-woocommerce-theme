Getting started

This guide will walk you through setting up and using your new WordPress theme to build your website. Even if you’ve never built a website before, WordPress makes it easy to create a professional and beautiful online presence.

What is WordPress?

WordPress is a popular platform used by millions of people to create websites. It’s known for its user-friendly interface and drag-and-drop functionality. This means you can create and customize your website without needing to write any code.

What is a Theme?

Think of a WordPress theme like a pre-designed website template. It provides the overall look and feel of your website, including layout, styles, and colors. Our theme offers a variety of customization options so you can tailor it to your specific needs and brand identity.

Benefits of Using Our WordPress Theme:

If you’re looking for a user-friendly and cost-effective way to build your website, WordPress is a great option. It’s suitable for individuals, businesses, and organizations of all sizes, with the flexibility to cater to various website needs.

If you are not familiar with WordPress, please get accustomed with it before proceeding to using our theme. Here are some links that will help you to become more confident with WordPress:

First Steps with WordPress: https://wordpress.org/support/article/first-steps-with-wordpress-classic/

Access a vast amount of resources, tutorials, and support from the extensive WordPress community. You can find helpful guides and forums at https://wordpress.org/support/welcome/

Installation

Theme Requirements

Before using Hitsxyz, you first need to make certain that your server and WordPress meet the theme’s requirements, including  WordPress new version. Most problems, such as white screen after theme installation, out of memory errors, fails when importing demo content, etc., are usually related to server settings and excessively low PHP configuration limits. You can change them by yourself, or contact your hosting provider and request the limits be increased to the following minimums:

  1. Php 7.4
  2. WordPress version 6.1.0
  3. Memory Limit 258mb
  4. Max execution time 180
  5. Max input var 3000
  6. Upload file size 35m

Use the WordPress server info plugin to check your PHP configuration limits.

Theme Installation

You can install the theme in either of two ways, but first, make sure your system meets the previously described requirements.

Method 1 – WordPress admin dashboard

  1. Unzip hitsxyz_v.x.x.zip
  2. Navigate to Appearance > Themes > Add New.
  3. Choose hitsxyz.zip file. Press the Install Now button to upload and install the theme.

If you get an error: “Are you sure you want to do this?”, it usually occurs when either the PHP memory limit or max post size for WordPress has been reached. See Hitsxyz Theme Requirements. The easiest workaround for this is to install Hitsxyz through FTP (see below). Alternatively, you’ll need to change several settings, either on WordPress, or on your web host.

Method 2 – FTP

  1. Connect to your server via FTP software.
  2. Unzip hitsxyz_v.x.x.zip and Extract hitsxyz.zip file and upload the extracted hitsxyz folder to /wp-content/themes/ folder on your server.
    Navigate to Appearance > Themes and activate Hitsxyz theme.

Child Theme

A WordPress child theme is a theme that inherits the functionality of the parent theme. With a child theme, you can make modifications or add additional functionality to the parent theme. The child theme offers the safest and easiest way to modify an existing theme, whether you want to make a few small changes or extensive ones.

Hitsxyz fully supports Child Theme.

One-Click installation

To make setting up your site easier and faster, you can import Demo Content.

This allows you to import pages, posts, products, slider, mega menu, custom blocks, theme settings, or additional pages from our demos.

We kindly request that you refrain from renaming the theme folder before importing the demo content. Additionally, it is advisable to deactivate any third-party plugins before initiating the import process.

To import Demo Content, follow these steps:

  1. Install and activate Hitsxyz theme.
  2. Then you will get a notification about installing the required plugins and Install the required plugin.
  3. Go to Appreance> Hitsxyz Importer and choose the demo you want to install. Press the Import button.
  4. Wait until you see the Success message. Check the frontend.
  5. Now you can regenerate demo images into the appropriate sizes for selected areas. Install the Regenerate Thumbnails plugin, navigate to Tools > Regenerate Thumbnails tab, and Regenerate All Thumbnails. This can take several minutes depending on the server speed and the number of images you have.
  6. Update permalink on Settings > Permalink and click save change.

The most intensive task in the demo import is the image import process, which takes the most time and server memory. So, you can do two things to solve this issue:

Change the default time of one AJAX call
The default value is 25 seconds. Add this code at the end of your theme functions.php file or using WPCode Snippets Plugin:

function ocdi_change_time_of_single_ajax_call() {
    return 10;
}
add_filter( 'ocdi/time_for_one_ajax_call', 'ocdi_change_time_of_single_ajax_call' );

Disable the generation of smaller images during the import.
While importing, smaller versions of images are being generated, which takes up a lot of server memory, so you can disable that in the plugin with a line of code. Add this code at the end of your theme functions.php or using WPCode Snippets Plugin:

add_filter( 'ocdi/regenerate_thumbnails_in_content_import', '__return_false' );

If the import is complete and you used the above solution, please install this plugin: Regenerate Thumbnails and run it in Tools > Regen. Thumbnails. This will then create the smaller versions of images, that we skipped in the import.

After the import, you should remove the added code from the functions.php file or deactivate on WPCode.

PRO TIPS:
  1. After install plugin, delete all page before import the demo. Because page from Woocommerce like Shop, Checkout, Cart will duplicate. If you miss this, go to Woocommerce -> Settings -> Advanced update Cart page, Checkout page, My account page and go to Woocommerce -> Settings -> Advanced update Shop Page.
  2. Compare product will not work properly, you need to update it by go to Yith -> Compare and click save.
  3. If you use cache plugin, and compare product still not working just wishlist this path on your cache plugin ?yith_wcqv_count ?action=yith-woocompare-add-product&id= ?action=yith-woocompare-view-table

Follow the steps below to update the theme:

  1. After you download the new theme package. Delete the current theme. Don’t worry about this. Your data won’t be lost.
  2. You upload the new theme package via WordPress or FTP software.

If you translated the theme to another language, you should backup your translation files in the themes > hitsxyz > languages folder before updating. You should also backup the translation files in plugins.

After updating the theme, you should update the recommended plugins if there is a notification about updating plugins. You go to Appearance > Install Plugins to update plugins. If you can not update the recommended plugins in wp admin, you should manually update them. You can find the plugin packages in the theme folder > framework > plugins folder.

Translations

Hitsxyz is ready to translate by using the po file. If you want to build a multilingual site, you need to use the WPML plugin. Our theme does not include the WPML plugin. It is only compatible with the WPML plugin. You have to buy a license for this plugin.

Using Po File

In this section, you will know how to build a multilingual website by using the WPML plugin. All the information needs to get started and use it is on the WPML.org page.

Below are some reference links:

  • WPML Documentation
  • Translating Widgets

Installing and Setting Up the WPML plugin

You need to install a recent version of WPML, including the String Translation, Translation Management and CMS Navigation modules. After installing these plugins, you set up Languages from WPML > Languages and translate your content.

Showing language switcher dropdown style on header

You go to WPML > Languages. You enable the Custom language switchers option. Then, you click the Customize button. A popup will appear. You change the Language switcher style option to Dropdown or Dropdown click.

Translating Pages and Posts

Go to Pages and pick a page for translation. Click on the plus icon. If the page is already translated, the plus icon will be replaced by the pen icon.

Translating Categories

To translate a category, click on the language which you want to translate it and then add a new category. Make sure the new category is linked to an original category.

Translating Menus

You go to Appearance > Menus. Just add a new menu. You select the language and the original menu which links to the new menu.

WPML can synchronize menus for you. This means that WPML can generate and keep in sync menus for other languages pointing to the translated versions of the pages, posts or categories.

Translating the theme and other texts

You go to WPML > String Translation. Find the text in the list and translate it. Please make sure you check the Translation is complete checkbox after translating.

Sometimes, some texts can not be found. You need to go to WPML > Theme and Plugins localization. You scan Strings in the themes. Then, you go back to the String Translation page and find those texts again.
